# Слобода

RTS про колонізацію Дикого поля (Слобожанщина, XVII ст.) —
результат експерименту зі створення RTS у стилі Warcraft за допомогою нейромереж.

## Технологии

- TypeScript + Vite
- Phaser 3
- Процедурная генерация графики (спрайты, текстуры, анимации — всё рисуется кодом, внешних ассетов нет)

## Запуск

### Локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Сборка кладётся в `docs/` — GitHub Pages раздаёт её с ветки `main`.
