# СЛОБОДА RTS — контекст для Claude Code

RTS про колонізацію Дикого поля (Слобожанщина, XVII ст.).
Стек: TypeScript + Vite + Phaser 3. `npm run dev` → localhost:5173. `npx tsc --noEmit` для типів.

## Походження і мапінг

Базовий движок (WC2-подібна RTS): A* pathfinding, туман війни, економіка
gold/lumber+food(popcap), будівництво, AI, формації, каравани, StoryController (DSL сценаріїв).

**Внутрішні id рас збережені** (на них зав'язані шляхи арту в public/assets/art):
- `alliance` = Слобожани, `horde` = Татари (презентаційний шар у RACE_LABEL/labelByRace)

Юніти (внутрішній id → лейбл → арт):
- worker → Селянин → арт селянина з сокирою (є work-анімація)
- footman → Козак → арт отамана з шаблею
- archer → Стрілець → арт козака з мушкетом
- knight → Сердюк → арт характерника (тимчасово; характерник стане героєм у M4)
- catapult → Гармата → процедурний арт движка (замінити пізніше)

Будівлі: townhall→Хутір, farm→Хата, barracks→Курінь, workshop→Кузня, tower→Дозорна вежа.

## Asset pipeline (scripts/)

- `clean_assets.py` — видаляє запечений сірий фон-шахматку з AI-генерованих PNG:
  кластеризація кольорів периметра → flood від країв → фільтр дрібних острівців
  (іскорки-вотермарки) → ерозія 1px проти ореолу. Опції: --tol --resize --frames --island
- `build_unit_sheets.py` — збирає з 4-кадрових листів Слободи (stand/windup/strike/hit)
  спрайтшити формату движка: 192×192, ряди = напрямки S/E/N/W; walk/death синтезуються
  (боб/прогресивне падіння). Згенеровані листи лежать у public/assets/art/units/alliance/.

Чисті вихідні ассети Слободи: public/assets/sloboda/{objects,units,portraits}
(об'єкти 256px — для M2/M3/M5: лендмарки, солеварня, городище, чумаки, стан).

## Стан проєкту

Зроблено (M0+M1): ребрендинг (палітра степу, лейбли, меню), стартовий загін з бійцями
(SKIRMISH_CONFIG.start.fighters: 3 козаки + 1 сердюк), інтеграція юніт-арту Слободи.

Зроблено (M2): економіка на 4 ресурси + населення. Economy: gold/lumber/salt/food +
pop/popCap (хати/хутір дають ліміт), EconCost-об'єкти, missing(). ResourceNode: 'salt'
(соляне джерело 2×2, текстура з assets/sloboda) та 'food' (туша, псується за
RESOURCE.carcassRotMs). Полювання: Animal (deer/boar, бродять біля лісу, тікають від
кривдника), ПКМ по звіру = полювання, смерть → onAnimalKilled → туша → селяни
розбирають → деплоять їжу в хутір. Спавн: spawnSaltDeposits/spawnWildlife у скірміші.
UI: бар Золото/Дерево/Сіль/Їжа/Люди, іконки salt/food, тултіпи джерела/туші/звіра.
Гачок для вартості в їжі готовий (cost.food в EconCost), але юніти поки їжу не коштують —
щоб не ламати AI, який не полює; вмикати у Сценарії 1 через story-економіку.
